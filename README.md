🚀 Built My First Cybersecurity Tool – Credential Leak Scanner

I recently built a small cybersecurity web application that checks whether an email address appears in known public data breaches.

🔐 **Credential Leak Scanner**

This tool allows users to:
• Check if their email appears in public breach databases
• View the number of breaches associated with the email
• Understand their potential risk level
• Get basic security recommendations to protect their accounts

🛠 **Tech Stack**
• Python
• Flask
• REST API integration
• Bootstrap (responsive UI)
• Render (deployment)
